## ✨ Features
- **3D 공간 구성**: 입체적인 방 구조와 감성적인 조명 효과로 몰입도 높은 시뮬레이션 제공
- **드래그 앤 드롭 배치**: 원하는 가구를 클릭만으로 배치하고 회전 및 이동 가능
- **로그인 없이 사용 가능**: 누구나 쉽고 빠르게 시작 가능, SNS 로그인 시 저장/공유 기능 제공
- **간편한 조작법 & 감성적인 UI**: 튜토리얼을 통해 누구나 1분 내 익숙해질 수 있는 인터페이스
- **공간 저장 및 공유 기능**: 배치한 공간을 캡처하거나 공유하여 아이디어 확산 가능
